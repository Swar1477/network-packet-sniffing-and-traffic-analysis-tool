# Network Packet Sniffer

This project is a Python-based network packet sniffer developed using the Scapy library.

The tool captures network packets from a selected interface and allows basic traffic analysis.

Features

Capture packets from network interface  
Enable promiscuous mode  
Filter packets by protocol  
Log packet information to a file  

Technologies Used

Python  
Scapy  
Linux Networking  

Usage

Run the script with root privileges:

sudo python3 packet_sniffer.py

Learning Outcome

Understanding of packet capture  
Network traffic monitoring  
Basic packet analysis
